# Personal academic homepage

Minimal Jekyll site based on [Jon Barron's website](https://jonbarron.info/)
(via [leonidk's Jekyll fork](https://github.com/leonidk/leonidk.github.io)).

**If you're continuing this in Claude Code, read [`CLAUDE.md`](./CLAUDE.md) first** —
it has the project state, the list of placeholders to fill, the schema for adding
papers, and local-preview / deploy instructions.

Quick start:
```bash
bundle install
bundle exec jekyll serve   # http://127.0.0.1:4000
```
